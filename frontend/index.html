About

Structural Response Prediction Using ML — ML-powered structural performance prediction.

school

Research Background

This application is developed as part of an M.Sc. thesis focused on predicting structural performance of RC buildings using machine learning.

map

Data Source

Based on over 400 RC structure building data from Bangladesh. Results accuracy may vary according to location.

smart_toy

ML Models

Three Gaussian Process Regression (GPR) models custom-trained from scratch on the collected dataset. Each model uses a Matern 1.5 kernel and was preprocessed with PowerTransformer and StandardScaler to predict column overstress, story drift, and story sway respectively.

trackpad_target

Model Performance

Story Sway R² = 0.958  |  Story Drift R² = 0.926  |  Column Overstress R² = 0.930

construction

Technology Stack

dns

Backend Framework

FastAPI — Modern, fast web framework for building APIs with Python 3.8+ based on standard Python type hints. Handles all prediction and calculation endpoints.

web

Frontend

Streamlit — Powers the web UI, embeds the FastAPI frontend in an iframe. Tailwind CSS provides utility-first styling with a custom design system. Chart.js renders interactive doughnut charts for risk visualization.

neurology

Machine Learning

scikit-learn — Gaussian Process Regression with Matern 1.5 kernel. PowerTransformer applies Yeo-Johnson transformation for normality. StandardScaler standardizes features. Models achieve R² > 0.92 across all three targets.

table

Data Processing

pandas & NumPy — Data manipulation and numerical computing. Training dataset of 411 RC building records with 14 input features processed through the full ML pipeline.

deployed_code

Model Serialization

joblib — Efficient serialization of scikit-learn pipelines including PowerTransformer, StandardScaler, and GPR models into compact .joblib and .pkl files (~7.6 MB each).
